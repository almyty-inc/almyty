# Skills Gateway

Drop your tools into any AI coding agent as installable skills. A Skills gateway renders each tool as a `SKILL.md` file following the [Agent Skills](https://agentskills.io) specification. The almyty Skills CLI installs and auto-injects those files into 30+ agents, including Cursor, Windsurf, Claude Desktop, and GitHub Copilot.

![Gateway detail page](/screenshots/gateway-detail.png)

## Create the gateway

1. Open **Gateways** in the sidebar and click **Create Gateway**
2. Select **Skills** as the protocol

![Create Gateway dialog](/screenshots/gateways-create-dialog.png)

3. Enter a name, slug, and optional description
4. Click **Create**

Assign tools from the **Tools** tab on the gateway detail page. Each assigned tool becomes one `SKILL.md` file. Copy the gateway slug: you will use it with the Skills CLI.

![Assigning tools on the gateway Tools tab](/screenshots/gateway-detail-tools.png)

## Install with the Skills CLI

Install the gateway's skills into your local agents:

```bash
npx @almyty/skills install @acme/my-skills
```

Authenticate first if needed:

```bash
npx @almyty/skills login
npx @almyty/skills login --token your-api-key
```

### Watch mode

Poll the gateway every 30 seconds and refresh SKILL.md files when tools change:

```bash
npx @almyty/skills watch @acme/my-skills
```

Stop with Ctrl+C or:

```bash
npx @almyty/skills stop
```

### List and remove

```bash
npx @almyty/skills list
npx @almyty/skills remove @acme/my-skills
```

The CLI writes SKILL.md files to each agent's skill directory and registers them for auto-loading.

## SKILL.md format

Each tool produces a markdown file like this:

```markdown
---
name: get_users
description: Retrieve a list of users with pagination
endpoint: https://api.almyty.com/utcp/acme/my-skills/tools/get_users/invoke
method: POST
auth: bearer
---

# get_users

Retrieve a list of users with pagination.

## Parameters

| Name  | Type    | Required | Description              |
|-------|---------|----------|--------------------------|
| page  | integer | no       | Page number              |
| limit | integer | no       | Items per page (default: 20) |
```
