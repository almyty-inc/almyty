# Skills Gateway

The Skills gateway generates `SKILL.md` files following the [Agent Skills](https://agentskills.io) specification. The almyty Skills CLI installs and auto-injects these files into 30+ AI coding agents, including Cursor, Windsurf, Claude Desktop, and GitHub Copilot.

![Gateways page](/screenshots/gateways-empty.png)

## In the UI

1. Navigate to **Gateways** in the sidebar
2. Click **Create Gateway**
3. Select **Skills** as the protocol

![Create Gateway dialog](/screenshots/gateways-create-dialog.png)

4. Enter a name, slug, and optional description
5. Click **Create**
6. Assign tools from the **Tools** tab on the gateway detail page

![Gateway tools](/screenshots/gateway-detail-tools.png)

7. Copy the gateway slug -- you will use it with the Skills CLI

## Via the API

### Create a gateway

```bash
curl -X POST /gateways \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Skills Gateway",
    "type": "skills",
    "endpoint": "/my-skills"
  }'
```

### View generated SKILL.md files

```bash
curl /gateways/{id}/skills \
  -H "Authorization: Bearer $TOKEN"
```

The response contains the rendered SKILL.md content for every tool assigned to the gateway.

## SKILL.md format

Each tool produces a markdown file like this:

```markdown
---
name: get_users
description: Retrieve a list of users with pagination
endpoint: https://your-instance/utcp/acme/my-skills/tools/get_users/invoke
method: POST
auth: bearer
---

# get_users

Retrieve a list of users with pagination.

## Parameters

| Name  | Type    | Required | Description              |
|-------|---------|----------|--------------------------|
| page  | integer | no       | Page number              |
| limit | integer | no       | Items per page (default: 20) |
```

## Skills CLI

Install and manage skills from the terminal.

### Install

```bash
npx @almyty/skills install @acme/my-skills
```

### Watch mode

Polls the gateway every 30 seconds and updates SKILL.md files when tools change.

```bash
npx @almyty/skills watch @acme/my-skills
```

Stop with Ctrl+C or:

```bash
npx @almyty/skills stop
```

### List and remove

```bash
npx @almyty/skills list
npx @almyty/skills remove @acme/my-skills
```

### Authentication

```bash
npx @almyty/skills login
npx @almyty/skills login --token your-api-key
```

## Supported agents

| Category | Agents |
|----------|--------|
| **IDE** | Cursor, Windsurf, VS Code (Copilot), JetBrains |
| **AI Assistants** | Claude Desktop, ChatGPT, Gemini |
| **Dev Tools** | GitHub Copilot CLI, Aider, Continue, Cody |
| **Frameworks** | LangChain, LlamaIndex, AutoGPT, CrewAI |
| **Others** | 20+ additional agents |

The CLI writes SKILL.md files to each agent's skill directory and registers them for auto-loading.
